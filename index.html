🛡️ Server Admin - Game Rules

1. Griefing

Griefing or promoting the act of griefing is strictly prohibited.

"to grief": to ruin other players' experiences.

Minor Griefing

  • Building on top of everyone's builds in the server
  • Putting spikes around the spawn area

Result of Action: None (can be prevented by server owners)

Standard Griefing

  • A build that kills all players upon spawning (e.g., Teleport Traps, Microwaves)

Result of Action: 14d ban

Major Griefing

  • Any build that makes the server unplayable (e.g., server crashers, laggers)

Result of Action: Permanent ban

2. NSFW

Minor Breakage

  • An upside down T
  • A build slightly suggestive

Result of Action: 3d ban

Major Breakage

  • Building a detailed NSFW build
  • Playing NSFW audios
  • Hosting servers dedicated to showing NSFW

Result of Action: Permanent ban

3. Exploiting

Minor Breakage

  • Using third party tools for little to no advantage in the game

Result of Action: None

Major Breakage

  • Using executors to inject code into the game to grief servers (e.g., flinging everyone)

Result of Action: Permanent ban

4. Destructively abusing glitches

  • E.g., abusing a glitch to be able to kick/ban moderators from your server

Result of Action: 7d ban → Permanent on 2nd offense

5. Degrading the game's reputation

  • Criticism is allowed, as long as it isn't destructive and makes others not want to play the game (e.g., spreading misinformation)

Result of Action: Permanent ban

6. Roblox TOS

  • Follow Roblox TOS at all times
