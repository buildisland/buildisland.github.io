What is Build Island Reborn?

Build Island got shut down. Bild Iland appeared but you had to join their group just to play and the mod team was corrupt. So Saintfire built something better.

🏝️

Open Servers

No group required. Open a beacon, share your server with the world, and let anyone join. Just like the original Build Island was always meant to be.

🛡️

Real Moderation

Build logs, trusted ranks, vote kick, and an anti-grief system that actually works. The chaos that killed the original Build Island won't happen here.

🔧

Actively Updated

New features, bug fixes, and improvements ship regularly. Not once a year. Not never. A game that's actually being looked after.

⚡

Wiring System

Connect blocks together to build machines, traps, vehicles, games within games. The wiring system lets your builds actually do things.

💰

Stamps Currency

Earn Stamps just by playing. Spend them on server boosts, save slots, and cosmetics. A global economy built into the game from day one.

📦

Modkits & Imports

Import community modkits to add hundreds of new blocks, weapons, gear, and tools to your server. Supports saving your Bild Iland builds.

Everything Build Island should've had

Reborn keeps everything you loved and adds the stuff Bereza never got around to.

⚡

Wiring System

Connect blocks with wires to build machines, doors, traps, and full game mechanics. Includes logic gates, number variables, string variables, boolean variables, array variables, timers, and more.

Original Feature
Logic Gates Variables Timers Sensors
🪙

Stamps Currency

Earn Stamps just by playing — $50 every 5 minutes plus a bonus for having friends in your server. Spend them on server boosts, extra save slots, and more. No pay-to-win.

New in Reborn
Earn by Playing Friends Bonus Send to Players
🚀

Beacon Boosting

Spend Stamps to boost your server to the top of the server browser for 3 hours. Boosted servers have a pink name and auto-beacon every 5 minutes so you never have to manually bump.

New in Reborn
Auto-Beacon Pink Title Top of List
🛡️

Anti-Grief System

Build logs track every block placed and deleted. Trusted ranks let owners control who can build freely. Vote kick handles disruptive players when the owner isn't around. The NSFW chaos that killed the original Build Island won't happen here.

New in Reborn
Build Logs Trusted Ranks Vote Kick Moderation
📡

Webhook Block

Wire a Webhook Message Sender block to anything in your build — a kill detector, a door, a button — and get instant Discord notifications. Whitelist-only for security.

New in Reborn
Discord Integration Wirable URL Whitelist
🧱

Block Placer

A wiring block that places any block by AssetId at a configurable position, rotation, and size when triggered. Build machines that build things. Automated construction.

New in Reborn
Wirable Any AssetId Custom Size
💾

Cloud Save Slots

Save your builds to the cloud and load them into any server. Start with 3 free slots and buy more with Stamps. Publish your builds publicly or whitelist specific players.

Original Feature
Cloud Saves Public Builds Whitelist Buy with Stamps
📦

Modkit Imports

Import community-made modkits to add hundreds of new blocks, weapons, NPCs, vehicles, and decorations to your server. Includes classic modkits like Liberty, Tekk3, Starburst, and more.

Original Feature
Weapons NPCs Vehicles Decor
🌐

URL Fetcher Block

Fetch live data from whitelisted URLs and pipe it into your wiring system. Pull from Pastebin, GitHub Gists, or Roblox APIs to create dynamic builds powered by real external data.

New in Reborn
Pastebin GitHub Roblox API Whitelist-Only
👥

Friends Panel

See which of your friends are playing and jump straight into their server from the main menu. No hunting through the server browser. Just click and join.

Original Feature
Friend Detection 1-Click Join

Custom Chat Tags

Rank-Based Coloured Tags Custom Bubbles
